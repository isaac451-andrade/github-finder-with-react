# GitHub Finder

Um buscador de perfis do GitHub desenvolvido para colocar em prática conceitos avançados de **React** com **TypeScript**. O projeto consome a API do GitHub para retornar informações de usuários e seus repositórios mais recentes.

## Tecnologias

Este projeto foi construído utilizando as seguintes ferramentas:

* **[React](https://reactjs.org/)** (com Vite)
* **[TypeScript](https://www.typescriptlang.org/)** (Tipagem forte e segurança)
* **[React Router Dom](https://reactrouter.com/)** (Navegação via `createBrowserRouter`)
* **[React Icons](https://react-icons.github.io/react-icons/)** (Ícones de UI e Spinners)
* **CSS Modules** (Estilização isolada e organizada)

## Funcionalidades

- [x] Busca de usuários via API do GitHub.
- [x] Exibição de dados do perfil (Avatar, Seguidores, Seguindo e Localização).
- [x] Listagem dos melhores repositórios do usuário.
- [x] Sistema de rotas dinâmicas para visualização detalhada.
- [x] Feedback visual de carregamento (Loader) e tratamento de erros (User not found).
